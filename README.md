I decided to work on this specific project because of my recent interest in Myers-Briggs Type Indicator
Test (MBTI Test). It is a personality categorization test specifically on Sensing/Intuition,
Thinking/Feeling, Judging/Perceiving). This test was originally developed by Katharine Cook
Briggs and her daughter Isabel Briggs Myers during WWII, it is based on Carl Jung's theory of
Psychological types according to Google.

This project will focus on my knowledge of Python and to provide users with a fun experience in
learning about their personality type. I would like to put my knowledge of Python programming
into a real project that I might encounter real problems in the real world. What I have learnt so
far such as control flows, arithmetic, etc. will be applied in this project.
